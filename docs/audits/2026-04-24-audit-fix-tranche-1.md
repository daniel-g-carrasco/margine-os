# Margine Public Audit Fix Tranche 1 - 2026-04-24

## Fixed

- Removed the Rofi baseline path from package manifests, provisioning payloads,
  host sync inventory, Hyprland binds, and launcher fallback logic.
- Kept Walker as the primary launcher and Fuzzel as the only official fallback.
- Made `validate-runtime-baseline` actionable and fixed its Hyprland screenshot
  bind detection for `conf.d` based configurations.
- Added `--product` and `--flavor` support to `validate-runtime-baseline`.
- Fixed the manifest de-duplication helper that could exit early under
  `set -e`.
- Reworked local AUR override staging so root stages local patch assets for the
  AUR build user.
- Changed the AUR build-user pacman sudo rule from permanent to temporary and
  validated it with `visudo`.
- Added `__pycache__/` and Python bytecode ignore coverage, then removed
  generated bytecode from the home payload.
- Converted the main update, boot, provisioning, and recovery user-facing output
  touched by this tranche to English.
- Added a root validator check for ESP random-seed world-accessible permissions.
- Changed the launcher wrapper to prefer package-owned `/usr/bin/walker`; an
  explicit `MARGINE_WALKER_BIN` can still override it.
